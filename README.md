# streamlit-dashboards
Streamlit Cognite dashboard examples

## Streamlit links:

https://streamlit.io/


## Env Setup:

```angular2html

pyenv virtualenv venv
pyenv activate venv
pip install -r requirements.txt

```

Change utils/.env.example file name to .env and add in env variables

## Run Instructions:

```angular2html
streamlit run dashboard.py 
```
